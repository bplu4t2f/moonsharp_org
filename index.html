    • Moon#

    An interpreter for a very close cousin of the Lua language, written entirely in C# for the .NET, Mono, Xamarin and Unity3D platforms.

    Project written&maintained by Marco Mastropaolo Hosted on GitHub Pages — Theme by mattgraham

    Project Status

    The project has just been started, yet it is able to parse most Lua structures and execute them correctly.

    Version 0.1.9 has been rewritten using a VM for faster interpretation and is available for download.

    Roadmap

    • support for all core language structures (see documentation for differences between Moon# and Lua)
    • optimizations
    • better integration between Lua/Moon# tables and CLR objects
    • debugger embeddable in applications using Moon#
    • REPL interpreter
    • standard library
