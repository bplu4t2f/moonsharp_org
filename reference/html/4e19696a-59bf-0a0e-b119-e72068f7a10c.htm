Script
This class implements a Moon# scripting session. Multiple Script objects can coexist in the same program but cannot share data among themselves except through the _UNIVERSE table. Data accessed through the _UNIVERSE table is thread safe, so Script objects can execute in different threads with no issue, as long as all their callbacks are aware and compatible.
