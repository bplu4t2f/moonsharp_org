MoonSharpModuleMethodAttribute Class
In a module type, mark methods or fields with this attribute to have them exposed as module functions. Methods must have the signature "public static DynValue ...(ScriptExecutionContextCallbackArguments)". Fields must be static or const strings, with an anonymous Lua function inside. See MoonSharpModuleAttribute for more information about modules.
