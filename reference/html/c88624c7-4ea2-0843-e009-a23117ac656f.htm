IPlatformAccessor Interface
Interface to abstract all accesses made to the underlying platform (OS, framework) by the scripting engine. Can be used both to support "non-standard" platforms (i.e. non-posix, non-windows) and/or to sandbox the behaviour of the scripting engine. It's recommended that no class implement IPlatformAccessor directly, and rather extend PlatformAccessorBase.
