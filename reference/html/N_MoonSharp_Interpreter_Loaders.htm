MoonSharp.Interpreter.Loaders Namespace
Namespace containing classes used to customized how scripts are loaded from external files.
Classes
  ClassDescription
EmbeddedResourcesScriptLoader
A script loader loading scripts from an assembly resources
FileSystemScriptLoader
A script loader loading scripts directly from the file system (does not go through platform object)
ScriptLoaderBase
A base implementation of IScriptLoader, offering resolution of module names.
UnityAssetsScriptLoader
A script loader which can load scripts from assets in Unity3D. Scripts should be saved as .txt files in a subdirectory of Assets/Resources. When MoonSharp is activated on Unity3D and the default script loader is used, scripts should be saved as .txt files in Assets/Resources/MoonSharp/Scripts.
Interfaces
  InterfaceDescription
IScriptLoader
Class dictating how requests to read scripts from files are handled. It's recommended that no class implement IScriptLoader directly, and rather extend ScriptLoaderBase.
