MoonSharp.Interpreter.Interop.LuaStateInterop Namespace
Namespace containing classes used to provide a minimal support for porting code based on the classic "LuaState" model (e.g. code based on KopiLua, UniLua or Lua itself).

LuaBase
Classes using the classic interface should inherit from this class. This class defines only static methods and is really meant to be used only from C# and not other .NET languages. For easier operation they should also define: using ptrdiff_t = System.Int32; using lua_Integer = System.Int32; using LUA_INTFRM_T = System.Int64; using UNSIGNED_LUA_INTFRM_T = System.UInt64;
